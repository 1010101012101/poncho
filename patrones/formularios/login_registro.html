Contraseña (para registro)

Sirve para que el usuario pueda acceder a una plataforma con información o acciones personalizadas. Si se trata de una nueva plataforma se debe usar Mi Argentina y no desarrollar un sistema nuevo de login. Más información sobre Mi Argentina.

Funcionamiento

Siempre deben mostrarse las reglas de contraseña antes de que el usuario pueda escribirla.

No pedir al usuario que escriba la contraseña 2 veces (la plataforma debe tener una forma de recuperar la contraseña a través de correo electrónico).

Contraseña (para ingresar)

Sirve para que, una vez registrado, el usuario pueda ingresar a una plataforma. La diferencia con el campo de contraseña para registrar, es que el de ingresar no valida si el usuario escribió algo que no siga las reglas de contraseña.

Funcionamiento

Siempre deben mostrarse las reglas de contraseña antes de que el usuario pueda escribirla.

No pedir al usuario que escriba la contraseña 2 veces (la plataforma debe tener una forma de recuperar la contraseña a través de correo electrónico).
